## Got Burger

**Halal burger restaurant in Ypsilanti, MI — full website and AI phone caller.**

- 3D spinning burger hero (React Three Fiber, Draco-compressed 7.9MB → 715KB, 91% reduction)
- Menu system — 10 categories, sticky tabs, per-item customizer, add-to-cart
- Stripe-powered online ordering — no DoorDash commission
- Location page — Google Maps, hours, Halal badge
- In-store display board — full-screen, auto-rotates every 12 seconds
- Print-ready menu PDF layout
- SEO targeting "halal burger Ypsilanti" and "halal fast food Ann Arbor"
- GitHub Actions CI — auto Draco compression on new GLB files
- AI Phone Caller (in progress) — VAPI → Square Orders API → kitchen display

**Stack**: Next.js, React Three Fiber, Tailwind CSS, Zustand, Stripe, VAPI, Square API, Vercel
